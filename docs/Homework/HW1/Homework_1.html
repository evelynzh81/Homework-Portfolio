Part j (7 points)

Generalize this by writing a function that will do this for a given team, and create a tibble with this information for all teams. Arrange this tibble first by winning percentage (descending) and by team (alphabetically) if there are any ties when it comes to winning percentage. A for loop and the add_row() function may be useful here.

win_perc = function(x) {
  df <- cbbga6 |>
  mutate(
      result = case_when(
      home == x & score_difference > 0 ~ 1,
      home == x & score_difference < 0 ~ 0,
      visitor == x & score_difference < 0 ~ 1,
      visitor == x & score_difference > 0 ~ 0,
      TRUE ~ NA_real_
    )
  )

answer = df |>
  summarize(
    team = x,
    win = sum(result == 1, na.rm = TRUE),
    loss = sum(result == 0, na.rm = TRUE),
    win_perc = win / (win + loss)
  )
return(answer)
}

Note, this is a little challenging. The most straightforward way to do this is to use the pivot_longer() function with cols = the names of the two columns that have the away and the home teams (the names of these columns should not be in quotes, but they should be in a vector c()). Try to do this yourself first, but using AI for some help is fine.

cbbga8 = cbbga7 |>
  filter(home=='Utah Tech'|
           home=="Southern Utah" |
           visitor == "Utah Tech"|
           visitor == "Southern Utah")

points_df <- cbbga8 |>
  select(home, visitor, home_score, visitor_score) |>
  pivot_longer(
    cols = c(home, visitor),
    names_to = "side",
    values_to = "school"
  ) |>
  mutate(
    points = ifelse(side == "home", home_score, visitor_score)
  ) |>
  filter(school %in% c("Utah Tech", "Southern Utah")) |>
  select(school, points)

ggbox(points_df, school, as.numeric(points))
